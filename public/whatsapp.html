📖 Como Funciona

1. O CallMeBot permite enviar mensagens WhatsApp gratuitamente (limite de 100/dia).

2. Para ativar, envie "I allow callmebot to send me messages" para +34 644 52 53 00 no WhatsApp.

3. Você receberá uma API Key como esta: 1748366

4. Cole o número e a API Key acima e clique em Salvar.

5. O TrayBot responderá automaticamente perguntas sobre pedidos, estoque e vendas!
